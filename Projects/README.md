# Projects

This directory contains the source code for the major, integrated projects developed during the course. 

## Contents

- **Project Implementations:** The main VHDL design files (e.g., `projeto.vhd`). These files typically represent complex digital systems that combine multiple structural components, finite state machines (FSMs), and datapath elements.

## Execution

Projects in this directory often depend on the fundamental components located in the `Operators` folder. When setting up your simulation or synthesis environment, make sure to include all necessary dependencies and compile the packages beforehand. Refer to the corresponding PDF guidelines in the `Assignments` folder for the exact project specifications.
