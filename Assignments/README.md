# Assignments

This directory contains the problem statements, instructions, and guidelines for the laboratory exercises and projects.

## Contents

- **Lab Instructions:** PDF files (e.g., `cap5_1s24.pdf`, `lab08_25.pdf`, `lab11_25.pdf`) detailing the objectives, circuit specifications, and tasks to be completed for each laboratory session.
- **Project Guidelines:** PDF files (e.g., `projeto1s25*.pdf`) describing the requirements and specifications for the final project.

Refer to these documents to understand the expected functionality of the VHDL code located in the `Solutions` and `Projects` directories.
