# Demonstrations

This directory contains screenshots and visual evidence of the VHDL code execution and simulation.

## Contents

- **Simulation Waveforms:** Screenshots showing the timing diagrams and behavior of the circuits during simulation.
- **RTL Viewer / Technology Map:** Screenshots demonstrating the synthesized hardware structures.
- **Compilation Results:** Evidence of successful compilation and synthesis without critical errors.

These files serve as verification that the implemented VHDL modules and labs behave as expected according to their theoretical truth tables and problem statements.
