# Solutions

This directory contains the completed VHDL implementations for the assigned laboratory exercises.

## Contents

- **Lab Implementations:** VHDL files (e.g., `lab02_2.vhd`, `lab04.vhd`, `lab06.vhd`, `Lab07.vhd`, `lab9.vhd`, `lab10.vhd`) that provide the solutions to the tasks described in the `Assignments` folder.
- **Testbenches:** VHDL testbench files (e.g., `lab04_tb.vhd`) used to simulate and verify the correctness of the lab implementations. 

## Structure

The solutions often instantiate the fundamental components found in the `Operators` directory. Make sure the required packages are compiled before attempting to synthesize or simulate these lab solutions.
